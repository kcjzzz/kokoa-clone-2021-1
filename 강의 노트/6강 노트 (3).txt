◆ 6.27 Settings Screen part One

단축명령어
header.alt-header>div.alt-header__column*3
태그명.클래스명 (자식선택자) 태그명.클래스명 *개수

    <header class="alt-header">
        <div class="alt-header__column"></div>
        <div class="alt-header__column"></div>
        <div class="alt-header__column"></div>
    </header>


justify-content: space-between; 은
글자수에 따라서 중앙 정렬이 완벽하게 안 될 수 있음.

따라서 확실하게 중앙 정렬을 하려면 약간의 편법을 쓰는 것

(예를 들어
alt-header__column이 가로축으로 3개의 클래스가 있을때 이걸 중앙정렬 하려면

.alt-header__column:first-child {
  margin-right: auto;
}
.alt-header__column:last-child {
  margin-left: auto;
}

이렇게 첫번째와 마지막(세번째) 마진값을 각각
right: auto 와 left: auto;로 주면 완벽하게 중앙정렬이 된다.

대신 부모 클래스에 display:flex와 align-items와 justify-content는 center로 맞춰놓아야됨.
